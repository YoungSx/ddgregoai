# 注册流程

## 流程概览

```
OAuth URL → 注册页 → 邮箱/密码 → 个人信息 → [验证/授权] → 完成
```

## 详细步骤

### 1. 生成 OAuth URL

使用 Sub2API 生成授权链接，记录返回的 `session_id`。

### 2. 打开 OAuth URL

浏览器打开生成的 URL。

### 3. 点击注册

在授权页面点击"注册"链接。

### 4. 输入邮箱

输入一个可用的 DDG 邮箱（格式：`word-word-word@duck.com`）。

**注意**：如果提示"账户已存在"，换一个邮箱重试。

### 5. 输入密码

输入密码后继续。

### 6. 填写个人信息

- 姓名：任意英文姓名
- 生日：18-60岁范围内
- 点击确认

### 7. 检查页面

此时可能出现两种情况：

| 页面 | 处理方式 |
|------|----------|
| 邮箱验证页 | 见 [VERIFY.md](./VERIFY.md) |
| 电话验证页 | 见 [LOGIN.md](./LOGIN.md) |

### 8. 完成授权

邮箱验证通过后，会出现授权确认页。点击"继续"。

### 9. 回调服务器

点击授权后，OAuth 会重定向到 `localhost:1455`，回调服务器会：
- 显示 code 和 state
- 提供创建账号表单

### 10. 完成

在回调页面输入 session_id，点击"创建账号"。

账号创建成功！可以在 Sub2API 管理后台查看。
